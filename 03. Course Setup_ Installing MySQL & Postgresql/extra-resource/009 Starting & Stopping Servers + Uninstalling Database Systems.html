In the previous lectures, we installed MySQL & Postgresql.

Here are summaries of common tasks you might want to perform at some point:

  • Stop a running MySQL server: 

    • MySQL: Windows | macOS | Linux

    • Postgresql: Windows | macOS | Linux

  • Start a MySQL server:

    • MySQL: Windows | macOS | Linux

    • Postgresql: Windows | macOS | Linux

  • Update Postgresql to a newer version:

    • MySQL: Windows | macOS | Linux

    • Postgresql: Windows | macOS | Linux

  • Uninstall MySQL:

    • MySQL: Windows | macOS | Linux

    • Postgresql: Windows | macOS | Linux

  • Reset the root user password:

    • MySQL: Windows | macOS | Linux

    • Postgresql: Windows | macOS | Linux

  • Switch to the legacy password flow (MySQL only): Windows | macOS | Linux

Also have a look at the official documentation pages for full details:

  • MySQL Official Documentation: https://dev.mysql.com/doc/

  • Postgresql Official Documentation: https://www.postgresql.org/docs/
